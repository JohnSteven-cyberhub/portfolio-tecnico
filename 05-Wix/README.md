# 🌐 Creación de Páginas Web con Wix

## Descripción

Demostración de habilidades en **creación y diseño de sitios web** utilizando la plataforma **Wix**, incluyendo personalización de plantillas, gestión de contenido y configuración de elementos interactivos.

---

## 🧪 Proyectos Realizados

### Proyecto 1 — Sitio Web Personal / Portafolio
- **Tipo**: Página personal / CV online
- **Características**:
  - Sección "Sobre mí" con foto y descripción profesional
  - Sección de habilidades y experiencia
  - Formulario de contacto funcional
  - Diseño responsivo (adaptado a móvil y desktop)

### Proyecto 2 — Landing Page de Negocio *(ejemplo)*
- **Tipo**: Página de servicios
- **Características**:
  - Header con llamada a la acción (CTA)
  - Sección de servicios con íconos
  - Galería de imágenes
  - Integración de Google Maps
  - Botón de WhatsApp

---

## 🛠️ Funcionalidades de Wix Utilizadas

| Funcionalidad | Descripción |
|---------------|-------------|
| Editor drag & drop | Colocación libre de elementos en la página |
| Plantillas | Punto de partida para diseño profesional |
| Wix App Market | Integración de apps (chat, formularios, etc.) |
| SEO básico | Configuración de meta títulos y descripciones |
| Dominio personalizado | Configuración de dominio propio |
| Wix Mobile Editor | Ajuste del diseño para dispositivos móviles |
| Formulario de contacto | Recepción de mensajes directamente al correo |
| Galería de imágenes | Presentación visual de proyectos o productos |

---

## 📐 Proceso de Creación

```
1. Definir objetivo del sitio (¿qué quiero comunicar?)
2. Seleccionar plantilla adecuada al tipo de negocio/perfil
3. Personalizar colores, tipografía e imágenes según identidad visual
4. Agregar y organizar el contenido (texto, imágenes, videos)
5. Configurar el menú de navegación
6. Revisar versión móvil y ajustar elementos
7. Configurar SEO básico (título, descripción, URL amigable)
8. Publicar el sitio
```
